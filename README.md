# Risk of Rain 2 custom texts
## Installation

**1. Find the `Languages` folder in the ROR2 files**
For the game installed through Steam it's in `C:\Program Files (x86)\Steam\steamapps\common\Risk of Rain 2\Risk of Rain 2_Data\StreamingAssets\Language`. There will be folder with `.json` files for each language the game has.

**2. Replace the `en` folder**
Just replace the `en` folder with the one from this repository, and you are good to go
